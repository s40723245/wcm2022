Nginx

設定 nginx, 讓其中所伺服的檔案只允許系上電腦下載

worker_processes  1;
events {
    worker_connections  1024;
}
http {
    include       mime.types;
    default_type  application/octet-stream;
    sendfile        on;
    keepalive_timeout  65;

    server {
        listen       [::]:80 ipv6only=off;
        listen       80;
        server_name  localhost;
        
     
        #allow 140.130.XX.0/24;
        allow 2001:288:6004:XX::0/32;
        deny all;

        #access_log  logs/host.access.log  main;
        

        location / {
            root Y:/tmp/wcm2022/;
        }

        location /tmp/ {
            root   Y:/;
        }
        autoindex on;
    }
}

因為已設定允許系上的網路ip可連上，因此用自己的網路時需要把deny all註解掉，否則只能用系上電腦才能連上nginx伺服器
